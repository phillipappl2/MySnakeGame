## Description
- Beat based snake game rogue like
---
## Features
### Movement
- to the beat
	- 60, 120, 180, 240, 300, 360 
	- base is 180, long snake 60, about to die 360
- the longer the snake is, the slower it moves
### Currency
- apples -> more snake segments
- snake segments -> powerups, skills
- coins?
### Shop
- powerups -> in levels
- permanent upgrades -> in base
### Base
- safe zone where we go to after beating a level
- choosing levels
### Skills
- speed -> longer snake can move faster
- health -> more body segments
### Powerups
- drop snake segment upon pressing a key (escape strategy)
### Enemies
- other snakes
- when snake hits other snake 1 body part is lost
- RNG drops?
- defeat a snake, get his segments
### Death
- resets whole progress
- permanent bonuses?
### Levels
#### Types of levels
- maze
#### Procedural generation of level
